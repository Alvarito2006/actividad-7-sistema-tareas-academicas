## 8. Dificultad encontrada

Una de las principales dificultades encontradas fue que no todos los integrantes tenían comunicación directa entre sí desde el inicio. En particular, se presentó la situación de no contar con contacto directo con Angee, por lo que fue necesario recurrir a Nora para establecer la comunicación y confirmar la integración del equipo.

Además, al tratarse de una actividad en entorno remoto, fue necesario coordinar la información mediante mensajes y confirmaciones progresivas.

---

## 9. Estrategia aplicada para resolver el problema

Para resolver la dificultad de comunicación, se aplicó la siguiente estrategia:

1. Se contactó primero a Nora para proponer el equipo.
2. Se explicó claramente el mini-proyecto que se deseaba trabajar.
3. Nora sirvió de apoyo para avisar a Angee.
4. Luego se compartió el repositorio con Diego para integrarlo al trabajo.
5. Finalmente se organizó el contenido del proyecto y se recopilaron las evidencias en el repositorio.

Esta estrategia permitió mantener la continuidad del trabajo y asegurar la participación del grupo.
